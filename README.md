# NLP-copilot

Stores mistakes and mistake attributes in JSON for API call.

## Installations

```
pip install language-tool-python
pip install flask
pip install flask-sqlalchemy
pip install flask-mysqldb
```

## How to Run

In MySQL CLI or WorkBench:

```
CREATE DATABASE nlp;
```

In terminal:

```
cd flask_app
flask run
```
